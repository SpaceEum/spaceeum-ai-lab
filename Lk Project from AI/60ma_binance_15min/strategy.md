# 60기간 신고가 돌파 매매 전략

## 1. 기본 설정

-   **거래소:** 바이낸스 USDT 선물 (Binance USDT-M Futures)
-   **타임프레임:** 15분 (15-minute)
-   **레버리지:** 10배 (10x)
-   **마진 모드:** 격리 (Isolated)
-   **주요 지표:** 60기간 단순이동평균선 (60-period Simple Moving Average, 이하 `MA(60)`)

## 2. 자금 관리

-   **진입 금액:** 전체 자산의 5%를 포지션 진입에 사용합니다.
-   **최소 주문 수량:** 만약 계산된 진입 금액이 해당 페어의 최소 주문 금액보다 작을 경우, 최소 주문 금액으로 진입합니다.
-   **손절 (Stop Loss):** 진입 가격 기준 **-3%** 지점에 설정합니다.
-   **트레일링 스탑 (Trailing Stop):**
    -   **활성화 시점:** 포지션 진입 후, 가격이 익절 방향으로 **3%** 이상 상승했을 때 활성화됩니다.
    -   **콜백 비율 (Callback Rate):** 3%로 설정합니다. (고점 대비 3% 하락 시 익절)

## 3. 종목 선정

### 3.1. 실시간 거래 대상 선정 (`Trade.py`에서 사용)

실시간 거래에서는 아래 **2가지 조건을 모두 만족**하는 티커를 매수 기회를 포착할 대상으로 선정합니다.
1.  **거래량 상위 500:** 24시간 거래량(USDT 기준) 순위가 상위 500위 이내인 티커

2.  **역배열 5기간 지속:** 아래 **'역배열 조건'이 최근 5개 캔들 연속으로 충족**된 티커
    -   **일목균형표 지표 정의 (기본값: 9, 26, 52):**
        -   `전환선 (Tenkan-sen)`: (최근 9기간 최고가 + 최근 9기간 최저가) / 2
        -   `기준선 (Kijun-sen)`: (최근 26기간 최고가 + 최근 26기간 최저가) / 2
        -   `후행스팬 (Chikou Span)`: 현재 캔들의 종가를 26기간 뒤로 이동시킨 값
        -   `선행스팬 1 (Senkou Span A)`: ((전환선 + 기준선) / 2)를 26기간 앞으로 이동시킨 값
        -   `선행스팬 2 (Senkou Span B)`: ((최근 52기간 최고가 + 최근 52기간 최저가) / 2)를 26기간 앞으로 이동시킨 값

    -   **역배열 조건 (AND 조건):**
        1.  `현재 종가 < 선행스팬 1 < 선행스팬 2` (현재 가격이 하락 추세의 구름대 아래에 위치)
        2.  `현재 종가 < MA(60)`
        3.  `전환선 < 기준선`
        4.  `기준선 < 선행스팬 1`
        5.  `후행스팬 < 26기간 전 종가` (현재 가격이 26기간 전 가격보다 낮음)

-   **선정 종목 저장:** 위 '선정 조건'을 만족하는 티커를 `Revers_order_(날짜_시간).db` 파일에 저장하여 실시간 거래 스크립트(`Trade.py`)가 모니터링하도록 합니다.
    -   **DB 컬럼:** `ticker`, `condition_met_time`

### 3.2. 백테스트 대상 선정 (`backtest_01.py`에서 사용)

백테스트에서는 전략의 잠재적 성능을 폭넓게 검증하기 위해, **거래량 상위 500개 티커 전체**를 대상으로 합니다.

-   **선정 조건:** 24시간 거래량(USDT 기준) 순위가 상위 500위 이내인 티커

## 4. 매매 전략 (Long Only)

### 4.1. 진입 조건 (Entry)

아래 **5가지 조건이 모두 충족**될 때 다음 캔들 시작가에 **지정가(Limit) 또는 시장가(Market)로 매수** 포지션에 진입합니다.

> **참고:** 실시간 거래(`Trade.py`)에서는 `Revers_order_*.db`에 저장된 티커를 대상으로 이 진입 조건을 감시하며, 포지션 진입 성공 시 해당 티커를 DB에서 삭제합니다.

1.  **MA(60) 위에 위치:** 현재 캔들의 종가가 `MA(60)` 위에 있음
    -   `[조건 상세] 현재 캔들 종가 > 현재 캔들 MA(60)`

2.  **60기간 신고가:** 현재 캔들의 종가가 최근 60개 캔들(현재 캔들 포함)의 최고 종가를 경신
    -   `[조건 상세] 현재 종가 == Highest(종가, 60)`

3.  **MA 상승 추세:** `MA(60)`이 상승 추세에 있음
    -   `[조건 상세] 현재  캔들 MA(60) > 이전 캔들 MA(60)`

4.  **60기간 모멘텀:** 현재 캔들의 종가가 60개 캔들 전의 종가보다 높음
    -   `[조건 상세] 현재 종가 > 60개 캔들 전 종가`

5.  **26기간 이격도 모멘텀:** 현재 캔들의 종가가 26개 캔들 전의 `MA(60)` 값보다 높음
    -   `[조건 상세] 현재 종가 > 26개 캔들 전 MA(60)`

### 4.2. 청산 조건 (Exit)

포지션을 보유 중일 때, 아래 조건 중 **하나라도 먼저 충족**되면 포지션을 청산합니다.

1.  **선행스팬 2 하향 돌파 (전략 청산):** 현재 캔들의 종가가 `선행스팬 2`를 하향 돌파
    -   `Crossunder(현재 종가, 선행스팬 2)`
    -   `[조건 상세] 이전 캔들 종가 >= 이전 캔들 선행스팬 2 AND 현재 캔들 종가 < 현재 캔들 선행스팬 2`

2.  **손절 (Stop Loss):** 가격이 진입가 대비 -3%에 도달

3.  **트레일링 스탑 (Trailing Stop):** 트레일링 스탑 조건 충족 시 익절

## 7. 실행 방법

-   **실행 모드:** 스크립트 실행 시, 사용자는 두 가지 모드 중 하나를 선택할 수 있습니다.
    1.  **자동 실행 (1번 선택):** 선택 시, 스크립트는 지정된 스케줄에 따라 자동으로 실행됩니다.
        -   **스케줄:** 매일 오전 `09:00`에 전체 프로세스(종목 선정 및 백테스트)가 실행됩니다.
    2.  **수동 실행 (2번 선택):** 선택 시, 스크립트는 즉시 1회 실행됩니다.

# Project Conventions

## Folder Structure
- **indicator/**: All indicator-related files (scripts, modules) must be placed here.
- **backtest/**: All backtest-related files (scripts, results, logs) must be placed here.

## Naming Conventions
- **Backtest Scripts**: Must follow the pattern `Backtest_[StrategyName].py`.
- **Backtest Output Files**:
  - CSV Results: `Backtest_[StrategyName]_results.csv`
  - Text Summary: `Backtest_[StrategyName]_summary.txt`
